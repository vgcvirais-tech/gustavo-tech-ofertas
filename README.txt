ARQUIVOS
- index.html: conteúdo da página
- style.css: aparência do site
- script.js: botão fixo e rastreamento
- assets/perfil.jpg: sua imagem

PUBLICAR NA VERCEL
1. Envie todos os arquivos para o repositório do GitHub.
2. Na Vercel, clique em Add New > Project.
3. Importe o repositório gustavo-tech-ofertas.
4. Em Framework Preset, deixe Other.
5. Não preencha Build Command.
6. Clique em Deploy.

TROCAR O LINK DO WHATSAPP
Procure no index.html por:
https://chat.whatsapp.com/

TROCAR A FOTO
Substitua assets/perfil.jpg por outra imagem com o mesmo nome.

OBSERVAÇÃO
A página é mobile-first e usa uma URL gratuita da Vercel quando publicada.
